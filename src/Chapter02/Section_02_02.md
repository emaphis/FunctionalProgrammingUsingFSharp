# Section 2.2 Operator precedence and association

## Arithmetic operators

    + unary plus
    - unary minus
    -------------------
    + addition
    - subtraction
    * multiplication
    / division
    % modulo (remainder)
    ** exponentiation

## Operator precedence and association

    Operator        Association
    --------        -----------
    **              Associates to the right
    * / %           Associates to the left
    + -             Associates to the left
    = <> > >= < <=  No association
    &&              Associates to the left
    ||              Associates to the left
